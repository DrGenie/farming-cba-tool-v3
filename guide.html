How to use the tool

  1. Download the template (Data tab) and enter one row per replicate plot.
  2. Mark the control by setting is_control to TRUE for the control rows.
  3. Upload your file (Data tab) then check the data status box (rows, treatments, controls, readiness).
  4. Run analysis (Results tab). The leaderboard compares treatment averages against the control average.
  5. Export (Export tab) to download the cleaned data and report tables.
Better structured data gives better analysis. If some optional fields are missing, the tool still runs and clearly labels results as indicative.

What each tab does

Home explains the workflow and minimum data needed.

Data is where you validate, upload, and fix common file issues. It also shows a preview and lets you download a cleaned file.

Results shows the treatment leaderboard, the plain language interpretation, and detailed tables. Replicates can be viewed under “Show more”.

Export provides one click downloads: report tables, cleaned data, and copy ready prompts for policy briefs.

Minimum data required

  • At least one control row where is_control is TRUE.
  • At least one treatment row where is_control is FALSE.
  • These columns must be present (exact spelling): treatment_name, is_control, yield_t_ha, total_cost_per_ha.

How the calculations work (with the example dataset)

This section shows every calculation step the tool performs using example_data.tsv and the default settings:

  • Price per tonne P = $500
  • Years T = 10
  • Discount rate r = 5%

Step 2. Discount factor (time value of money)

The tool uses an annual discount factor across the chosen years:

DF(r,T) = Σ[ t=1..T ] 1 / (1+r)^t

With r = 0.05 and T = 10, each year term is:

Year t 1/(1+r)^t
1 0.952381
2 0.907029
3 0.863838
4 0.822702
5 0.783526
6 0.746215
7 0.710681
8 0.676839
9 0.644609
10 0.613913

Sum of terms:

DF(0.05,10) = 7.721734929185

Step 3. Annual benefit from yield

Annual benefit per hectare is yield times price:

Annual benefit = mean_yield × P

Control

6.194611 × 500 = 3,097.31

Deep OM (CP1) + PAM

7.226651 × 500 = 3,613.33

Step 4. Present value of benefits and costs

Benefits and costs are treated as annual flows repeated over T years and discounted using DF:

PV benefits = annual_benefit × DF
PV costs = mean_cost × DF

Control PV benefits

3,097.31 × 7.721734929185 = 23,916.57

Control PV costs

694.627249 × 7.721734929185 = 5,363.73

Deep OM (CP1) + PAM PV benefits

3,613.33 × 7.721734929185 = 27,901.14

Deep OM (CP1) + PAM PV costs

884.882704 × 7.721734929185 = 6,832.83

Step 5. Net present value (NPV), benefit cost ratio (BCR), and comparison to control

NPV = PV benefits - PV costs
BCR = PV benefits / PV costs
Difference vs control (NPV) = NPV_treatment - NPV_control

Control NPV

23,916.57 - 5,363.73 = 18,552.84

Deep OM (CP1) + PAM NPV

27,901.14 - 6,832.83 = 21,068.31

Deep OM (CP1) + PAM difference vs control

21,068.31 - 18,552.84 = 2,515.47
